Features

Multi-Platform Support

Switch between MiniMax, GLM, DeepSeek, and Kimi. Each platform can be enabled or disabled independently.

Real-Time Monitoring

View daily and weekly usage percentages directly in your menu bar. Updates automatically.

Status Indicators

Dynamic color coding shows usage status at a glance. Green for healthy, yellow for warning, red for low.

Sparkle Updates

Automatic update notifications keep your app current with the latest features and improvements.

Dark Mode Support

Automatically adapts to your system appearance settings.

Bilingual Support

Available in English and Simplified Chinese.
